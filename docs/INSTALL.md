# 快速安装指南

## 第一步：生成图标

由于浏览器扩展需要图标文件，请按以下步骤操作：

1. 在浏览器中打开 `icon-generator.html` 文件
2. 选择您喜欢的图标样式和颜色
3. 点击"下载全部"按钮
4. 将下载的三个图标文件（icon16.png、icon48.png、icon128.png）移动到此文件夹

**或者**：您可以使用任何 16x16、48x48、128x128 的 PNG 图标，命名为：
- icon16.png
- icon48.png
- icon128.png

## 第二步：加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/` 并按回车
3. 打开右上角的 **"开发者模式"** 开关
4. 点击左上角的 **"加载已解压的扩展程序"** 按钮
5. 浏览并选择此文件夹（Moodle Downloader）
6. 扩展将立即安装并显示在列表中

## 第三步：开始使用

1. 访问 Monash Moodle：https://learning.monash.edu/my/
2. 进入任意课程页面
3. 点击浏览器工具栏中的扩展图标
4. 点击"扫描资源"按钮
5. 查看扫描结果，点击"一键下载全部"开始下载

## 常见问题

### Q: 扫描不到资源？
**A:** 确保您在 Moodle 课程页面上，并且页面已完全加载。

### Q: 下载失败？
**A:** 某些资源可能需要您先登录 Moodle。确保您已登录并且有访问权限。

### Q: 图标不显示？
**A:** 确保图标文件已正确放置在扩展文件夹中，并且文件名完全匹配（icon16.png、icon48.png、icon128.png）。

## 项目文件说明

- `manifest.json` - 扩展配置文件
- `popup.html` - 弹出窗口界面
- `popup.css` - 界面样式
- `popup.js` - 界面交互逻辑
- `content.js` - 页面内容脚本（扫描资源）
- `background.js` - 后台服务（处理下载）
- `icon*.png` - 扩展图标（需要生成）
- `icon-generator.html` - 图标生成工具
- `README.md` - 完整文档
